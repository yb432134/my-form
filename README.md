# Ismaaeel Meehan Arabic Institute Website

A professional, modern website for Ismaaeel Meehan's Arabic language teaching institute. The website features a clean design, responsive layout, and user-friendly interface to attract students and showcase Arabic learning services.

## Features

### 🎨 Modern Design
- Clean, professional layout with Islamic-inspired color scheme
- Beautiful typography using Inter and Amiri fonts
- Smooth animations and hover effects
- Mobile-first responsive design

### 📱 Responsive Layout
- Fully responsive across all devices (desktop, tablet, mobile)
- Mobile-friendly navigation with hamburger menu
- Optimized touch interactions for mobile users

### 🚀 Interactive Elements
- Smooth scrolling navigation
- Animated statistics counters
- Form validation and submission
- Notification system for user feedback
- Scroll-to-top functionality

### 📝 Content Sections
- **Hero Section**: Compelling introduction with call-to-action
- **Testimonials**: Real student reviews with star ratings
- **Services**: Comprehensive overview of teaching methodology
- **About**: Personal story and credentials
- **Contact**: FREE trial session form with detailed information
- **Footer**: Links, resources, and contact information

## Technical Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive functionality and animations
- **Font Awesome**: Professional icons
- **Google Fonts**: Custom typography (Inter + Amiri)

## Setup Instructions

1. **Download/Clone the files**
   ```bash
   # All files should be in the same directory
   - index.html
   - styles.css
   - script.js
   ```

2. **Open in Browser**
   - Double-click `index.html` to open in your default browser
   - Or serve via a local web server for best performance

3. **Customize Content**
   - Update contact information in the footer
   - Modify testimonials and services as needed
   - Replace placeholder social media links

## Customization

### Colors
The website uses a green color scheme that can be easily customized by modifying CSS variables:

```css
/* Primary Colors */
--primary-color: #2c5530;
--secondary-color: #4a7c59;
--accent-color: #f8fffe;
```

### Content Updates
- **Testimonials**: Update in the HTML testimonials section
- **Services**: Modify the services grid content
- **Contact Info**: Update footer contact details
- **Social Links**: Replace `#` with actual social media URLs
